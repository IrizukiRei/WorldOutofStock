# 世界、在庫切れ — ゲームルールブック

## 第一部：無くなったら困るもの遊び（消滅ゲーム）
### 副題：城を倒したら概念が死ぬ

## 1. 用意するもの
- プレイヤー2人（2国）
- 城っぽい置物2つ（🏯🏰でも紙切れでも指でも可）
- ペン（必須ではない。あると便利）

## 2. 目的
「無くなったら困るもの」を段階的に言い合い、最後にまとめて消滅させる。  
勝敗は基本なし。残るのはディストピアだけ。

## 3. 進行（5段階固定）
各段階で、各プレイヤーは「無くなったら困るもの」を1つ言う。短く、名詞で。

段階はこの順番で固定：
1. 人類に必要
2. 日本に必要
3. 京都に必要（ローカルは差し替え可）
4. 家に必要
5. かばんに必要（持たない派はポケット）

### ルール
- 前に言ったものの言い直し禁止
- 抽象でも具体でもOK（例：信頼／昼／抹茶／口紅／電気／布団）
- 「ないと困る」の基準はガバでよい

## 4. 仕上げ（消滅宣言）
5段階が終わったら、仕上げを行う。

- お互いの城（🏯🏰）をツンッと倒す（👉️）
- 各国が出した5つを読み上げる
- 宣言する：
  - 「○○帝国から、（5つ）が……＼消滅しました💥／」
  - 「△△帝国から、（5つ）が……＼消滅しました💥／」

### 効果
- それ以降、そのセッション中「消滅したもの」は存在しない扱い
- 会話・外交・言い訳・文化などで使えない

例：
- 昼が消えた国は、会談の終了時間が決まらない
- 口紅が消えた国は、唇カサカサ

## 5. オプション：要らないもの遊び（不要物版）
同じ手順で、今度は「不要」を言う。

1. 人類に不要
2. 日本に不要
3. 京都に不要
4. 家に不要
5. かばんに不要

仕上げも同じく消滅宣言。

### 効果（皮肉寄り）
不要物が消える＝困らないはずなのに、なぜか「それに依存していた言い訳」も消える。

例：
- 同情が消える → 泣き落とし外交ができない
- テレビが消える → プロパガンダを流せない
- 名刺が消える → 肩書きで殴れない

## 6. 平和ごっこゲームへの世襲（重要）
この消滅ゲームは、後半（平和ごっこ）の縛りになる。

### 世襲ルール（紙なし最短版）
- 各国、自分の「必要5つ」のうち、いちばん痛い1つを国家欠損に選ぶ
- 相手は1回だけ「それは痛くないやろ」と差し替え要求できる（1回のみ）
- 確定した国家欠損が、平和ごっこでの「空札（点が入らない色）」を決める

### 色の決め方（ざっくり）
- 体面・序列・権威・威圧・武勇・格 → 🟥誇りが空札
- 情報・交流・観光・通信・文化・メディア・ネット → 🟦信頼が空札
- 昼夜・時間・余裕・生活・体力・回復・眠り → 🟨暇が空札

## 第二部：世界、在庫切れ（カード版・完全版）
### カード版／5ターン／消滅世界世襲／拡張カセット対応

## 0. ゲームの芯
- この世界で実際に動くのは「空気」だけ
- 空気は3色：🟥誇り／🟦信頼／🟨暇
- 3色は「世界在庫」として共有管理（両国共通）
- 点数は2人で共通
- 両国の空気が一致（共鳴）した時だけ世界在庫が+1

## 1. 使うもの
- プレイヤー2人（2国）
- カード3種（実物がなくても宣言で可）
  - 🟥誇り
  - 🟦信頼
  - 🟨暇
- 5ターンを数える指

## 2. 世襲（消滅ゲームの欠損反映）
### 2-1
各国は消滅ゲームで自国から消えたものの中から「一番痛いもの」を1つ選ぶ。  
相手が「それ痛いな」と認めたら確定。

### 2-2
それを弱点色に変換する（基準は3つ）：
- 上下／体面／権威に関係 → 弱点：🟥誇り
- 交流／情報／文化に関係 → 弱点：🟦信頼
- 時間／余裕／生活に関係 → 弱点：🟨暇

### 2-3
弱点は「空札」になる。  
空札の色は出せるし言えるが、共鳴しても点にならない。

## 3. 進行（全5ターン固定）
- ターン1〜3：内政フェーズ（自分の3枚から出す）
- ターン4〜5：外交フェーズ（相手の過去3枚から借りて出す）

### 3-1. 事件（ターン3終了時に1回）
コイントス：
- 表：事件なし（ターン4は通常判定）
- 裏：ターン4だけ「ねじれ判定」
  - 🟥×🟥 → 🟦+1
  - 🟦×🟦 → 🟨+1
  - 🟨×🟨 → 🟥+1

※揃った色が空札なら+0。  
※拡張の「単独発火」「贔屓の共鳴扱い」はねじれ対象外（個別記述を優先）。

## 4. ターンのやり方（短文縛り）
各ターン、同時に1枚出し、短い一言を添える（長文禁止）。

例：
- 🟥誇り：「断固」「遺憾」「看過せん」
- 🟦信頼：「対話」「協調」「未来」
- 🟨暇：「注視」「保留」「検討」

※長文を言い出したら、そのターンは強制で🟨暇扱い。

## 5. 判定（共鳴ルール）
同じ色が揃った時だけ、その色が+1される。  
ただし、どちらかが空札なら+1は入らない。

- 🟥×🟥 → 誇り+1
- 🟦×🟦 → 信頼+1
- 🟨×🟨 → 暇+1
- 別色同士 → 変化なし

## 6. 終了（5ターン後の結末）
5ターン終了時、点が一番高い色がエンド。

- 誇り最多 → 開戦エンド
- 信頼最多 → 同盟エンド
- 暇最多 → 内部崩壊エンド
- 同点 → 白紙合意エンド

## 拡張カセット（追加ルール）
※通常は1本だけ差す想定。全部盛りも可能（その場合プレミアム自動発動）。

### A. 🔥発火パック（国内共鳴→単独発火）
#### A-1. 増え方
内政フェーズ（ターン1〜3）で🟥誇りを出すたび、その国の発火ゲージ+1（0〜3）。  
ただし🟥が空札の国は増えない。

#### A-2. 単独発火（ゲージ3）
ゲージ3の国は外交フェーズ（ターン4〜5）で🟥を出すと、
相手が🟥でなくても🟥共鳴成立扱い（誇り+1）。

#### A-3. リスク
単独発火中に信頼が0になったら、制裁・孤立エンド扱い。

### B. 💖贔屓パック（関係の特別扱い）
#### B-1. ON条件
外交フェーズ（ターン4〜5）で同じ相手色を2回とも借りたら贔屓フラグON。

#### B-2. 効果
ON中に1回だけ、別色でも共鳴扱いにして点を入れてよい。  
点はそのターンに自分が出した色へ。使用後はOFF。

### C. 🕌神話パック（物語化ボーナス）
#### C-1. 上がり方
同じ色の共鳴が2ターン連続で起きたら神話ゲージ+1（0〜2）。

#### C-2. 神話1
次のターン、共鳴成立時に+1点おまけ。

#### C-3. 神話2
神話2の間は、共鳴成立ターンのマイナス扱いを無視してよい。  
ただし空札による「点が入らない」は無視できない。

#### C-4. 下げ方
以下のどれかで神話ゲージを0に戻す：
- 🟨暇の共鳴が起きた
- 共鳴しないターンが2回続いた

### D. 🛖生存パック（生活余裕→破局回避）
#### D-1. 増え方
🟨暇の共鳴が起きたら生存+1（最大2）。

#### D-2. 効果（生存を1消費）
次のいずれかを1回だけ無効化できる：
- 🔥発火の制裁・孤立扱い
- 0到達などの急展開（運用している場合）
- そのターンのマイナスを全部打ち消す（プラスは残す）

## 全部盛り専用：プレミアム拡張（自動発動）
### P. 運命
※🔥発火＋💖贔屓＋🕌神話＋🛖生存を全部入れている時のみ有効。

### P-1. 自動発動条件（同時成立）
1. どちらかが🔥発火状態（発火ゲージ3）
2. 💖贔屓フラグON
3. 🕌神話ゲージ2
4. 🛖生存が1以上

### P-2. 効果（そのターンのみ）
そのターンの判定は色を無視して共鳴成立扱い。共鳴点は+2。

### P-3. 副作用（燃え尽き）
発動後にリセット：
- 神話ゲージ → 0
- 贔屓フラグ → OFF
- 生存 → 0
- 発火ゲージ → 1

### P-4. プレミアムエンド
運命発動ターンでゲーム終了した場合：
- 誇り最多 → 革命エンド
- 信頼最多 → 新秩序エンド
- 暇最多 → 無為の黄金期エンド

## 運用メモ（紙なし版）
- 基本だけなら何も問題ない
- 拡張1本差しはギリ脳内で回る
- 全部盛りは指カウント多めだが「運命」が盛り上がる
- 迷ったら弱点（空札）だけ決めて基本を1回回し、そこから拡張を差す
